Tribute Draft

June 2025

1. Abstract

A Tribute Draft is a reflection of a user's Acts of Consumption, recorded on the L2 Network. It is created by combining one or more of the user's Consumption Units (CUs) into a single unit, which is then created on L2 after the end of the World Wide Day.

2. Glossary

Term Definition
Coen

is a digital asset on Network whose value directly reflects Users' economic activities. Native Coins are minted strictly according to Emission Limit rules and are tradable freely on Exchanges.
Consumption Reflection Agent (CRA)

is a regulated entity responsible for retrieving, verifying, and posting transaction data. CRAs generate Consumption Units by aggregating a Consumption Records and confirming their legitimacy. They act as the bridge between traditional financial systems and the network's on-chain value creation.
Consumption Unit (CU)

is an entity representing a batch of Consumption Records aggregated daily per bank account. Each Consumption Unit qualifies as a whole for Tribute issuance and serves as the basis for Tribute allocation.
Settlement Token

is a Stable Token specifically approved through Governance Protocol for settlement purposes on Network. Network supports multiple Stable Tokens, but only a limited subset of these are designated as Settlement Tokens.
Tribute

is an option granting Users the right to mint Gratis, determined by the economic value captured from their Consumption Units. When a Tribute is Issued, it uses data from these Consumption Units to define specific conditions and attributes required for Gratis minting.
Wallet

is a digital tool on Network that securely stores and manages cryptographic keys. It doesn't hold digital assets themselves but provides Users with secure access to their assets on Network, enabling Users to send, receive, and manage Native Coins and Stable Tokens (including Settlement Tokens). Each Wallet manages a Private Key, which authorises transactions and generates the Public Key used to derive the Account Address.
Worldwide Day

is a 48-hour period that spans all global time zones, from UTC+14 to UTC−12. It is used as the standard temporal unit for synchronising Epochs, network operations, and global activity across the Network.

3. System Logic

The L2 Consumption Network is a secondary blockchain layer designed to handle high-throughput, manipulation-resistant reflection of user consumption behavior. It serves the CRA (Consumption Reflection Agent) community by offering an environment for submitting, validating, and consolidating Consumption Units (CUs) with a further logic of consolidation into Tribute Drafts — structured digital reflections of real-world user consumption.

This specialized L2 supports:

  • Submission and deduplication of Consumption Records
  • Creation and validation of Consumption Units
  • Generation of Tribute Drafts post-World Wide Day cycles.

CU is submitted on L2 by CRA, and checked on L2 for duplication. If CU is accepted by network it will be included in the Tribute Draft for the period of worldwide day reflecting consolidated user's consumption.

The event of CU validation and creation on L2 is considered as CU is accepted on the network and is ready for further consolidation in Tribute Draft.

Mechanics of CU creation and submission logic is defined in CRA: Consumption Unit Generation document.

The event of Tribute Draft validation and creation on L2 is considered as an consolidated record of user's consumption and is ready for submission for Wallet App in a form of original Tribute, then it has to be signed and sent via Wallet App to L1.

If the original Tribute is accepted by L1 and created on the network, the Tribute Draft is considered as utilized.

Object Status
Tribute Draft Created, Utilized

4. Attributes

Tribute Draft on the Network should have the following attributes:

Field Name Type Description Example
tribute_draft_id string Unique Tribute Draft identifier
created_at string ISO 8601 timestamp of when the Tribute Draft was created 2025-06-12T14:30:00Z
owner string User address on network 0x1234abcd5678ef901234abcd5678ef901234abcd
worldwide_date string Worldwide Day in ISO 8601 date format 2025-06-12
settlement_token string Settlement Token is used for account. EUR
settlement_amount_minor integer [10^6] Value of the Tribute in Settlement Tokens 12500000 (represents $12.50 if minor = 10^6)
nominal_qty_minor integer [10^18] Total nominal quantity in Coens 150000000 (represents 1.5 coen if minor = 10^8)
cu_hashes array Ordered list of CU hashes included in the Tribute Draft

6 Appendix

Tribute Draft Schema

tribute_draft:
  type: object
  description: Represents the input fields used to compute the hash of Tribute Draft
  required:
	- tribute_draft_id
    - created_at
    - owner
    - worldwide_date
    - settlement_token
    - settlement_amount_minor
    - nominal_qty_minor
    - state
    - cu_hashes

  properties:
  
  	tribute_draft_id:
      type: string
      description: Unique Tribute Draft identifier
      
    created_at:
      type: string
      format: date-time
      description: ISO 8601 timestamp of when the Tribute Draft was created and submitted to L2
      
    owner:
      type: string
      description: Network address associated with the owner
      
    worldwide_date:
      type: string
      format: date
      description: Worldwide Day in ISO 8601 date format


    settlement_token:
      type: string
      description: Token used for settlement (e.g., "USD", "EUR", "GBP", "CNY", "JPY", "HKD")

    settlement_amount_minor:
      type: integer [10^6]
      description: Value of the Tribute in Settlement Tokens
      example: 1200.67 tokens = 1200670000 (assuming 1 token = 1,000,000 microtokens)

     nominal_qty_minor:
       type: integer[10^18]
       description: Value of the Tribute Draft in Coens
       example: 200
       
     state:
     	type: string
      	enum:
        - accepted
        - utilized
      	description: State of the Tribute Draft

    cu_hashes:
      type: array
      items:
        type: string
      description: Ordered list of CU hashes included in the Tribute Draft
